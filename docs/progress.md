# Журнал прогресса

Хронология значимых изменений. Подробности — в истории git (`git log`).
Формат: от новых к старым.

## Июль 2026

**Паспорт камеры и инженерные сценарии**
- Отдельная страница-паспорт `camera.html?id=` — полноценный цифровой паспорт
  (HUD, характер, спеки, кликабельная эволюция, объективы, софт). Модалка стала превью.
- Прямые ссылки на фирменный софт в паспорте — проверенные официальные страницы
  Canon / Nikon / Sony / Fujifilm / Panasonic / OM System / Hasselblad / Leica / Pentax / Ricoh.
- Поле `mechanicalShutter` для всех камер — задел под стоп-моушен и предметную съёмку
  (7 камер только с электронным затвором: Z8, Z9, A9 III, R50 V, iPhone, DJI).

**Опросник — фундамент данных**
- Восстановлены флаги формата/типа из сенсора (1091 правка): полнокадровых стало 88 вместо 26.
- Убраны фантомные поля из скоринга (`lowLight`, `portrait`, `burstBuffer`), ложные правила
  (серия по числу слотов, Sony-хардкод). Отключён вопрос о битности RAW.
- Байонеты выверены: фикс-оптика (X100 и др.) обнулена, дальномерки Leica M восстановлены.

**Сообщество и админка**
- Фоторюкзак: добавление камер из камератеки, список в кабинете.
- Общая статистика фидбэка через Cloudflare Worker (только счётчики, без персональных данных).
- Админке добавлена вкладка «Фидбэк»; починен блокирующий запрос токена; рестайл под камератеку.

**Камератека и данные**
- Редизайн модалки в HUD-стиле Hasselblad; фильтры свёрнуты за кнопку.
- Цены переведены в рубли по вторичному рынку; удалены дубли; выверены автофокус и слоты.
- 4-уровневая сложность меню + новое поле эргономики, заведено в скоринг.
- Данные эволюции: `releaseYear`, `family`, `predecessorId` для всех камер.
- Массовый поиск картинок для объективов в админке.

**Основа проекта**
- Документы проекта переведены в `docs/*.md` как единый источник истины
  (Vision, Конституция, Свод).
- Единая навигация, футер, страницы-заглушки разделов, фундамент личного кабинета.
- Починена совместимость объективов по байонету (GFX, MFT, XCD, Z/RF/L).
